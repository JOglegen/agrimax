Want to customize the pages? Easy!

1. Update GitHub URL:
   Find: href="https://github.com/yourusername/agrimax"
   Replace with your actual GitHub URL
   
2. Update Company Name:
   Find: Record Harvest Enterprises
   Replace with your company name
   
3. Update Netlify Domain:
   Find: https://youragrimax.netlify.app

   In CSS section, update:
   --gn: #1e5631;  (main green)
